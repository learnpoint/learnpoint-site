On Thursday the 12th of June, Learnpoint will be updated with a new feature: Grading Scales. The feature makes it possible to customize existing scales, as well as adding new scales. Torsdagen den 12 juni kommer Learnpoint att uppdateras med en ny funktion: Betygsskalor. Funktionen gör det möjligt att anpassa befintliga betygsskalor och lägga till nya.

The feature will be particularly benificial for schools that run multiple forms of educations, because you can now use different scales for different courses. Funktionen kan vara särskilt användbar för skolor som bedriver olika typer av utbildningar. Nu kan du nämligen använda olika skalor i olika kurser.

If you have the role as School Administrator, you'll find the feature in the more menu: Om du har rollen Skoladministratör så hittar du funktionen i mer-menyn:

On the grading scales page, you can add new scales and customize existing ones: På sidan för betygsskalor kan du lägga till nya skalor eller redigera de som redan finns:

TIP: Grading steps are typically expressed in letters or numbers. But this feature also makes it possible to use emojis, which could be useful in some situations.
