Import sections Importera avsnitt

Karl West

April 4, 2024 4 april, 2024

The feature Import content makes it easy to reuse course content, but the feature have had a limitation. It wasn't possible to import a section without also importing course items from that section. Funktionen Importera innehåll används för att återanvända kursinnehåll, men funktionen har haft en begränsning. Det gick inte att importera ett avsnitt utan att även importera kursdelar från avsnittet.

This limitation is removed and it's now possible to import sections without having to import course items: Begränsningen är borttagen och nu går det att importera avsnitt utan att man behöver importera kursdelar:
